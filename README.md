## 安装

响应码部分依赖于 [jiannei/laravel-enum](https://github.com/Jiannei/laravel-enum)，需先进行安装

```shell
$ composer require jiannei/laravel-enum -vvv
```

再执行

```shell
$ composer require jiannei/laravel-response -vvv
```
